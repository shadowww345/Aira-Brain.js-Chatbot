# Aira Chatbot
**This is my first-ever chatbot, which I built about a year ago**

## Features
**- Knowledge Retrieval: Fetches information from Wikipedia, extracts the most relevant results, and saves them.**

**- Mathematical Operations: Capable of handling basic arithmetic (addition, subtraction, multiplication, division) and exponentiation.**

**- Natural Dialogue: Supports daily conversational interactions.**

### Note on Known Issues
**Disclaimer: Sometimes the bot might output gibberish like "okgfsd".**
**This usually happens when it encounters unknown information and tries to "predict" an answer, which leads to overlapping responses (early-stage AI hallucinations).**
